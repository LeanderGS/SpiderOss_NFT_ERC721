# SpiderOss_NFT_ERC721
Smart contract for the creation of a non-fungible token, ERC721(NFT).

· Development environment: RemixIDE

· Network: Ropsten

· Standar: ERC721 - OpenZeppelin

· Wallet: MetaMask

· Sistema de archivos interplanetarios (IPFS): Pinata Cloud
